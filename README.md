# Typing

Typing application powered by Spring Boot and React.

## Installation

Install node modules.

`$ yarn`

Compile web application.

`$ yarn build`

## Usage

`$ mvn spring-boot:run`

## Deployment

Maven package and skip test.

`$ mvn package -DskipTests`

Run jar.

`$ java -jar ./target/typing-0.0.1-SNAPSHOT.jar`
